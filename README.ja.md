# VR-megane-kaikan
[日本語のREADMEはこちらです: README.ja.md](README.ja.md)

虚想のメガネ展示室を体験できるVRアプリです。

## デモ
プロジェクトは https://code4fukui.github.io/vr-megane-kaikan/ で視聴できます。

## 機能
- 没入感のある360度の仮想環境
- メガネのショールーム設定を表示します

## 要件
このプロジェクトにはA-Frame、Web上のVR(仮想現実)体験を構築するためのフレームワークが必要です。A-Frameは提供された`index.html`ファイル内のCDNを介して含まれています。

## 使用方法
1. `index.html`ファイルをウェブブラウザで開きます。
2. ブラウザ内で直接VR体験を視聴できますが、より没入感のある体験をしたい場合はVRヘッドセットをお使いください。
